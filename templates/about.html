ElecVis for better energy usage

ElecVis is a new energy dashboard that is meant to create a more interactive environment for power users. The goal of this project is to help users to improve their energy information literacy.

Users can use the main page for submitting their CSV file which contains their energy data. They will be able to view their energy usage for whatever specific time period they choose. Data will be presented using line graph or Bar graph. Also, the tips section on the main page will show users their average monthly and daily usage. It will also provide the users with some tips that help reduce energy usage if needed.
